# 运行脑暴示例

在项目根目录执行。使用前确保 `.env` 中已配置 `FEISHU_WEBHOOK`。

---

## 1. 给男人卖胸罩

```bash
python3 -m brainstorm.run \
  --topic '如何向男性消费者有效沟通并销售胸罩（礼品/为伴侣购买等），形成可执行、有记忆点的方案' \
  --context 'briefs/men_bra_brainstorm.md'
```

或直接跑脚本：

```bash
bash run_brainstorm_men_bra.sh
```

---

## 2. 茶馆 × 潮牌跨界快闪

```bash
python3 -m brainstorm.run \
  --topic '在洛阳茶馆里办潮牌跨界快闪活动：低成本、让创作者被打动、引起更大范围用户注意' \
  --context 'briefs/luoyang_teahouse_crossover.md'
```

或直接跑脚本：

```bash
bash run_brainstorm.sh
```

---

说明：
- 如果需要推送到特定飞书群，在 `.env` 中设置 `FEISHU_WEBHOOK`
- 会话会写入 `runs/<时间戳>_session.md`，同时按轮推送到对应飞书群
