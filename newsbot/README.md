# 早知天下事 (`newsbot/`)

多源新闻采集 + AI 分析，自动生成结构化日报并推送到飞书群。

---

## 快速开始

```bash
# 最低配置：FEISHU_APP_ID + FEISHU_APP_SECRET + DEEPSEEK_API_KEY
python3 -m newsbot
```

启动后在飞书给机器人发消息即可使用。

**只想生成一次报告（不启动飞书长连接）：**

```bash
python3 -m newsbot --run
```

---

## 飞书命令

| 命令 | 说明 |
|------|------|
| `新闻` / `日报` | 手动触发今日新闻汇总 |
| `快报` | 精简版快速新闻摘要 |
| `科技新闻` / `财经新闻` | 按领域筛选 |

---

## 新闻来源

覆盖多个区域和语言：
- 华人圈 / 亚太 / 欧美 / 全球热点
- 自动去重、分段推送（适配飞书单条消息字数限制）

---

## 定时推送

默认每天 **08:00（北京时间）** 自动推送日报到飞书。

推送目标：
- 配置了 `FEISHU_WEBHOOK` → 推送到对应飞书群
- 配置了 `FEISHU_PUSH_OPEN_ID` → 推送给指定用户
- 都未配 → 推送给最近和机器人聊过天的用户

---

## 环境变量

**必填**：

| 变量 | 说明 |
|------|------|
| `FEISHU_APP_ID` | 飞书应用 App ID |
| `FEISHU_APP_SECRET` | 飞书应用 App Secret |
| `DEEPSEEK_API_KEY` | 新闻分析用的大模型 |

也可用专用凭证 `NEWSBOT_FEISHU_APP_ID` / `NEWSBOT_FEISHU_APP_SECRET`（同时跑多个 bot 时建议）。

**可选**：

| 变量 | 说明 |
|------|------|
| `FEISHU_WEBHOOK` | 飞书群 Webhook，用于群推送 |
| `TAVILY_API_KEY` | 提升新闻搜索质量（免费 1000 次/月） |

---

## 输出格式

日报按区域分段推送，每段包含：
- 新闻标题 + 来源
- AI 生成的一句话摘要
- 相关性和重要度标记

长报告自动拆分为多条消息，适配飞书字数限制。
